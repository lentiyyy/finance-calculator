## Описание

Веб-приложение для учёта личных доходов и расходов на чистом JavaScript (без фреймворков). Данные сохраняются локально в `localStorage` браузера и не покидают устройство пользователя.

## Возможности

- Добавление транзакций (доход/расход) с указанием суммы, категории, даты и комментария.
- Шесть предустановленных категорий: Еда, Транспорт, Жильё, Развлечения, Здоровье, Другие.
- Фильтрация по периоду (всё время / неделя / месяц / произвольный диапазон), типу, категории.
- Поиск по тексту комментария.
- Сводная статистика: суммарные доходы, расходы, баланс.
- Распределение по категориям с процентами и текстовым графиком (полосы из символов `#`).
- Цветовая дифференциация доходов и расходов.
- Экспорт отфильтрованных транзакций в CSV (UTF-8, разделитель `;`).
- Валидация ввода и уведомления об ошибках (toast).
- Адаптивная вёрстка от 320 px.

## Запуск

Файлы — статические, серверная часть не требуется. Достаточно открыть `app/index.html` через локальный веб-сервер (модули ES требуют HTTP/HTTPS, протокол `file://` их не загружает).

```bash
# Вариант 1. Python
cd app
python3 -m http.server 8000
# открыть http://localhost:8000

# Вариант 2. Node.js
npx serve app
```
